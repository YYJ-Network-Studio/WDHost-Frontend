Minecraft伺服器架設核心

以下內容適用Java伺服器。目前常見的伺服器架設核心有Spigot, Paper, PurPur，多數的插件都支援以上3款核心，本次教學以Paper示範，選擇的原因就不多介紹，有興趣的話請上網查。

原版伺服器架設教學

1. Minecraft需要依賴Java，因此請先安裝Javasudo apt install openjdk-8-jdk

給新手的小幫助：1.17以下版本使用Java 8即可，1.17使用Java 16，1.18使用Java 17。
點擊查看各Java版本的下載指令
  • 安裝Java 8：sudo apt install openjdk-8-jdk
  • 安裝Java 16：sudo apt install openjdk-16-jdk
  • 安裝Java 17：sudo apt install openjdk-17-jdk

    • 2. 下載伺服器核心wget 下載連結

    3. 執行伺服器核心java -Xmx1024M -Xms1024M -jar 伺服器核心名稱.jar

    給新手的小幫助：
    執行1.18.2java -Xmx1024M -Xms1024M -jar paper-1.18.2-379.jar -Dlog4j2.formatMsgNoLookups=true
    1024MB = 1GB以此類推，您也可以改成-Xmx1G -Xms1G
    防止Log4j漏洞請加上-Dlog4j2.formatMsgNoLookups=true
    請注意伺服器主機的RAM，假設您購買的伺服器為8G RAM，則分配給Minecraft伺服器的RAM必須小於8G RAM。

    4. 同意eula條款vi eula.txt將false更改為true

    給新手的小幫助：vi是Linux的文字編輯指令，執行指令後， 按下鍵盤的i即可開始更改內容。更改後，請按下鍵盤的esc並輸入:wq存檔即可。

    5. 再次執行伺服器核心java -Xmx1024M -Xms1024M -jar 伺服器核心名稱.jar

    6. 在Minecraft遊戲中輸入伺服器連線位置(IP)，即可連線至伺服器！

    每當SSH連線關閉Minecraft伺服器也會關閉，如何解決？

    只需要讓Minecraft伺服器在Screen裡執行即可。

    1. 首先，創建Screenscreen -S 名稱

    2. 進入指定的Screenscreen -r 名稱

    3. 進入指定的Screen後，您就可以執行Minecraft伺服器啟動的指令或.sh腳本

    4. 使用Ctrl + A + D快捷鍵離開Screen

    給新手小幫助：忘記Screen名稱的話，輸入screen -ls即可查看名稱和ID(您可以使用ID進入Screen)。
